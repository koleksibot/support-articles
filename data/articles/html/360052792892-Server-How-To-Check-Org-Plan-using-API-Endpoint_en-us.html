Use the below API endpoint to know your Organization plan on CircleCI Server. This will show you different parameters like trial end date, org name, etc.

 

If you would like to see license details, please check this endpoint or go to http://<fqdn>/admin/license.

 

<circleci_url>/api/v1/admin/organization/:username/plan

 

 

Note: You should be an admin to run this endpoint.

 

If you have any more questions, please submit a ticket to Support or write to support@circleci.com.
