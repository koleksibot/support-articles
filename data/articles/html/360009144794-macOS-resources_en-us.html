The macOS executor offers two resource_class levels:

- Medium: 4 vCPUs, 8GB RAM

- Large*: 8 vCPUs, 16GB RAM

*This resource size requires review by our account team. Feel free to open a support ticket if you would like to request access

Additional details and how to set a resource_class can be found in our configuration documentation.

Disk space is not dependent on the resource_class size and is instead affected by the Xcode version you are using. You can view the available disk space per version within our Xcode version documentation. Click the "Installed Software" link on the version you are using to get specifications.
