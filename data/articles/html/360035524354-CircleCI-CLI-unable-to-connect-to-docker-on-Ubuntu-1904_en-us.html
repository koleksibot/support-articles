Currently, we have an issue that 'circleci local execute' command is unable to connect to the docker on Ubuntu 19.04. It only happens when you install 'circleci cli' and 'docker' by using 'snap' command.

Snap has security features and 'circleci cli' seems to be restricted by this. Strict confinement uses security features of the Linux kernel, so if you change the permission of the commands or using the root account, it doesn't work. We are currently working on this issue. It will be fixed soon. 

Meanwhile, could you follow the instructions below?  

$ sudo snap remove circleci docker
$ sudo snap install --devmode circleci docker
$ circleci local execute --job build --config .circleci/config.yml
