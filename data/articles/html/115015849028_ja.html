CircleCI 2.0 ビルドの実行時に次のエラーメッセージが表示された場合

Cannot connect to the Docker daemon at unix:///var/run/docker.sock. 
Is the docker daemon running?

もしくは以下のエラーが表示された場合、

Exception message: /home/circleci/.docker/config.json (No such file or directory)

これは一般に、リモート Docker (setup_remote_docker) なしで Docker コマンドを実行しています。

CircleCI 2.0 で Docker コマンドを実行するには、次のドキュメントで説明しているように、setup_remote_docker ステップを追加する必要があります。 もしくは machine executorを利用することも可能です。
